# Hire Me
Hire Me gives you access to manny's network by providing a User Interface for hiring employees

# What does this mod do?
- Hire multiple employees at once
- Hire Employees to multiple properties at once
- Information Pop-up for when you hit the max employees for a property
- Error message when you go over your cash limit
- Easily delete employees if you spend to much
- Easily fire employees from different properties
- Easily transfer employees between properties
- UI saves the employees you selected after closing it
- Automatically minimzes properties that are at max capacity

# Where can I find this UI?
- [At the Warehouse showcased by MasakoTeam](https://www.youtube.com/watch?v=xET2MmgfDPI&lc=UgyCTund0Rs0gjZo0eJ4AaABAg.AKSe5E6mrW6AKWRTLyIZQV)
- Phone Application

# How to configure?
Phone application visiblity can be configured via the MelonPrefences.cfg file located under the UserData directory in the root game folder. In the file you will find an attribute labeled `EnablePhoneApp`. `true` enables the phone app and `false` disables the phone app

# Installation
1. Download and install [MelonLoader](https://github.com/LavaGang/MelonLoader/releases)
1. Download this mod.
1. Unzip the downloaded file
1. Make sure you select the correct mod for the branch of the game you have installed. Il2Cpp for main branch & Mono for alternate branch.
1. Drag the .dll into the "Mods" folder in your Schedule 1 files.
