# Version 2.0.0
- Hiring Interface shows currently hired employees
- Now you can fire employees from the UI
- Now you can transfer employees to different properties from the UI
- A Phone App that allows you to access the UI and manage employees remotely
- Configuration setting allowing you to disable the phone app if you dont want to use it.
- Mod is compatible with ModManager&PhoneApp mod by Prowiler
